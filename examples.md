# Garry – Examples

This file provides example uses of Garry in context: dialogues, narrative lines, and the canonical pangram analogue.

---

## 1. Basic Dialogue

### 1.1 Check-In

**A:** `gary?`  
**B:** `gaary… gary.`  

Interpretation:

- A: “You okay? / What’s up?”  
- B: “I’m not great, but I’ll be okay / I’m here.”

---

### 1.2 Soft Reassurance

**A:** `gary? gary?`  
**B:** `gaary. gary.`  

Interpretation:

- A: “Are we okay? Are we still aligned?”  
- B: “Yes, gently; we’re good.”

---

### 1.3 Boundary and Recovery

**A:** `gary! gary!`  
**B:** `GARY. gary…`  

Interpretation:

- A: “Push, push / Do it, do it.”  
- B: “Hard no / boundary; I know that lands heavy.”

---

## 2. Planning Something Risky

**A:** `gary? gary! gary.`  
**B:** `gaary… gary?`  
**A:** `GARY. gary.`  

Interpretation:

- A: `gary? gary! gary.`  
  - “Should we? → Yes, let’s. → Done.”

- B: `gaary… gary?`  
  - “I’m soft but uneasy; are you sure?”

- A: `GARY. gary.`  
  - “Decision’s final. We’ll normalize after.”

---

## 3. Indirect Affection

**A:** `gaary. gary… gaary.`  
**B:** `gaary. gary!`  

Interpretation:

- A: “I’m soft toward you, I’m awkward about it, but I still feel that softness.”
